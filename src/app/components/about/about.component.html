About AstroPartner

Project Purpose

AstroPartner is a web application developed as the final project for my Master’s degree in Web Development. It serves both an educational and exploratory purpose showcasing modern web technologies, external APIs, and user-centered design in a real-world application.

The application features lunar forecasts, a personal calendar, interaction forum, and astrology-based insights, built using Angular, a weather API for lunar data, and a custom backend that manages events, users, posts, comments, and daily horoscopes.

Why Astrology?

AstroPartner was inspired by a personal interest in astrology and the moon. Beyond its academic foundation, the project is a creative exploration—designed for those who enjoy cosmic themes and want practical tools built around them. Developing something both functional and meaningful made the entire process even more rewarding.

Credits & Acknowledgments

This application was made possible thanks to the following open-source content:

  • Weather API – Provides lunar phase data over a 14-day forecast period.
  • FullCalendar – Enables interactive scheduling and event visualization.
  • Angular Material – Responsive UI elements.

Additionally, AstroPartner includes original content stored in a custom backend, managing event data, user interactions, posts, comments, and daily horoscopes. Horoscope messages were sourced from the Perchance Horoscope Generator.
